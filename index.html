Откуда цены?Where do prices come from?

Цена = средневзвешенная стоимость input и output токенов по формуле: (3 × input + 1 × output) / 4. Веса 3:1 отражают типичное соотношение: промпт обычно длиннее ответа.Price = weighted average cost of input and output tokens: (3 × input + 1 × output) / 4. The 3:1 weights reflect a typical ratio: prompts are usually longer than responses.

Что исключено?What is excluded?

Reasoning-модели (o1, o3, o4-mini, DeepSeek-R1) исключены из регрессии. Причина: они генерируют скрытые «цепочки рассуждений» — тысячи токенов, которые пользователь не видит, но оплачивает. Сравнение их $/токен с обычными моделями некорректно.Reasoning models (o1, o3, o4-mini, DeepSeek-R1) are excluded from regression. Reason: they generate hidden "chains of thought" — thousands of tokens the user doesn't see but pays for. Comparing their $/token with regular models is incorrect.

Методология (как строятся графики)Methodology (how charts are built)

Следуем подходу Epoch AI:Following the Epoch AI approach:

1
Для каждого бенчмарка определяем «пороги качества» — это скор каждой модели, которая устанавливала новый рекорд производительности.For each benchmark, we define "quality thresholds" — the score of each model that set a new performance record.
2
Для каждого порога отслеживаем самую дешёвую модель, достигающую этого уровня, по мере выхода новых моделей.For each threshold, we track the cheapest model reaching that level as new models are released.
3
Получаем набор точек (дата, цена) — «за сколько можно было купить данный уровень качества в каждый момент времени».We get a set of data points (date, price) — "how much a given quality level cost at each point in time."
4
Строим лог-линейную регрессию: log₁₀(цена) = slope × год + intercept. Наклон (slope) определяет скорость удешевления.We fit a log-linear regression: log₁₀(price) = slope × year + intercept. The slope determines the rate of price decline.

Отрицательный наклон = цены падают. Значение 10|slope| — во сколько раз дешевле каждый год.A negative slope = prices are falling. The value 10|slope| = how many times cheaper each year.
