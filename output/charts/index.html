LLM Inference Price Trends

Воспроизведение и обновление анализа Epoch AI до марта 2026 года. Как быстро дешевеет доступ к LLM через API?

Главный вывод

Стоимость использования LLM через API падает экспоненциально. За тот же уровень качества ответов вы платите в среднем в 5 раз меньше каждый год. По некоторым задачам (наука, математика) — в 10–11 раз меньше.

Что это значит на практике? В 2023 году для задач уровня GPT-4 (GPQA ~50%) нужно было платить ~$15/млн токенов. В 2025 году тот же уровень качества стоит ~$0.17 (Gemini 2.0 Flash). Это в ~88 раз дешевле за 2 года.

Источники данных

Откуда цены?

Цена = средневзвешенная стоимость input и output токенов по формуле: (3 × input + 1 × output) / 4. Веса 3:1 отражают типичное соотношение: промпт обычно длиннее ответа.

  • Epoch AI / Artificial Analysis — основной источник для 66 исторических моделей (2021–2025). GitHub
  • Официальные страницы провайдеров — Anthropic, OpenAI, Google, Meta, DeepSeek, xAI, Mistral
  • ArtificialAnalysis.ai — перекрёстная верификация цен

Откуда бенчмарки?

  • MMLU — 57 предметов, от истории до физики. «Общие знания» модели. Максимум 100%.
  • GPQA Diamond — вопросы уровня PhD по физике, химии, биологии. Даже эксперты набирают ~65%. Максимум 100%.
  • HumanEval — 164 задачи на программирование (Python). Модель пишет код, тесты проверяют. Максимум 100%.
  • MATH-500 — 500 задач по математике (алгебра, геометрия, теория чисел). Максимум 100%.
  • MATH Level 5 — самые сложные задачи из MATH (олимпиадный уровень). Максимум 100%.
  • LMSys Chatbot Arena ELO — рейтинг на основе слепых сравнений пользователями. Шкала ELO (~1000–1400).

Источники скоров: официальные технические отчёты моделей, OpenAI simple-evals, Papers with Code, Epoch AI.

Что исключено?

Reasoning-модели (o1, o3, o4-mini, DeepSeek-R1) исключены из регрессии. Причина: они генерируют скрытые «цепочки рассуждений» — тысячи токенов, которые пользователь не видит, но оплачивает. Сравнение их $/токен с обычными моделями некорректно.

Методология (как строятся графики)

Следуем подходу Epoch AI:

1
Для каждого бенчмарка определяем «пороги качества» — это скор каждой модели, которая устанавливала новый рекорд производительности.
2
Для каждого порога отслеживаем самую дешёвую модель, достигающую этого уровня, по мере выхода новых моделей.
3
Получаем набор точек (дата, цена) — «за сколько можно было купить данный уровень качества в каждый момент времени».
4
Строим лог-линейную регрессию: log₁₀(цена) = slope × год + intercept. Наклон (slope) определяет скорость удешевления.

Отрицательный наклон = цены падают. Значение 10|slope| — во сколько раз дешевле каждый год.

Линия тренда

Все бенчмарки — линии регрессии Медиана: ~5x/год

Каждая линия — тренд для одного бенчмарка. Чем круче наклон — тем быстрее падают цены для этого типа задач.

GPQA Diamond (PhD-level Science) ~11x/год

Самое быстрое падение цен. В 2023 году для PhD-уровня нужен был GPT-4 Turbo за $15/MLN. В 2025 — Gemini 2.5 Flash за $0.85 с лучшим качеством.

MMLU (General Knowledge) ~4x/год

Самый «зрелый» бенчмарк — модели давно научились его решать. GPT-3 набирал 44% за $60, сейчас Phi 4 набирает 85% за $0.12.

MATH-500 ~10x/год

Математические задачи. Резкий прогресс: GPT-3.5 решал 44% задач, новые модели — 89%+, при этом в десятки раз дешевле.

HumanEval (Coding) ~4x/год

Задачи на написание кода. Бенчмарк сильно «насыщен» — топовые модели уже решают 90%+, поэтому дифференциация по цене менее выражена.

MATH Level 5 (Advanced Math) ~3x/год

Олимпиадная математика. Самый медленный тренд — только дорогие модели достигают высоких результатов, дешёвые пока отстают.

LMSys Chatbot Arena ELO ~6x/год

Рейтинг на основе предпочтений пользователей (слепые A/B тесты). Отражает «субъективное качество» модели. Шкала ELO, не проценты.

Интересные наблюдения

Opus подешевел в 3 раза

Anthropic снизила цену флагманской линейки Opus: с $15/$75 (Opus 4, май 2025) до $5/$25 (Opus 4.5/4.6, конец 2025 — начало 2026). Средневзвешенная цена упала с $30 до $10 за миллион токенов.

GPT-4.1 nano = Gemini 2.0 Flash по цене

Обе модели стоят $0.175/MLN при сильно различающихся бенчмарках. GPT-4.1 nano: MMLU 80.1, GPQA 50.3. Gemini 2.0 Flash: MMLU 76.4, GPQA 62.1. Выбор зависит от задачи.

Gemini 2.5 Flash — убийца по соотношению цена/качество

$0.85/MLN при GPQA Diamond 82.8% — это уровень GPT-4 Turbo ($15) за 18x дешевле, с значительно лучшим качеством.

DeepSeek-V3-0324 — рекордная плотность бенчмарков на доллар

$0.48/MLN при MMLU 88.5 и MATH Lv5 90.2. Ближайший конкурент по качеству (GPT-4.1) стоит в 7 раз дороже.

Пробелы в данных

Главная проблема — отсутствие MATH-500 и Arena ELO для новых моделей 2025–2026. Это занижает тренды для этих бенчмарков. С полными данными значения скорее всего были бы ближе к 9–40x Epoch AI.
